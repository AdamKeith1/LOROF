# LOROF
LOROF: Linux-on-RISCV-on-FPGA

Goal is to design, verify, and implement on FPGA a RISC-V RV32IMAC_Zicsr_Zifencei Sv32 Quad-Core Superscalar Out-of-Order CPU, successfully booting and running Linux.
